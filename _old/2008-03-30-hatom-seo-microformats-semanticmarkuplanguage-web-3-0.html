I was very sceptical about Semantic Search but looks like the web is moving to that direction.
Microsoft adopted hAtom Microformat for his new WebSlice feature in IE8, Yahoo announced they had began to use microformats on its shopping search engine kelkoo.

Can this be the beginning of a new era? Can it be the web 3.0?
